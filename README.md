
# Dimes – Unity Prototype

This repository contains the Unity prototype for the game **Dimes**

Dimes is a conceptual puzzle game where the player begins in a **one-dimensional environment** and gradually unlocks higher spatial dimensions.

## Prototype Features

The current prototype demonstrates the 1D stage of the game:

• Constrained player movement along a single axis  
• Beacon guidance system  
• Camera follow system  

## Scripts

Player.cs  
Handles player movement and restricts movement to the Z-axis.

CameraFollow.cs  
Controls the camera position behind the player.

BeaconPulse.cs  
Animates the beacon to pulse and guide the player.
